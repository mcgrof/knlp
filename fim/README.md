# FIM research scaffold

This subtree is the reproducible open-source home for FIM-guided research code
that does **not** belong in Marin itself.

Use this tree for:
- paper-oriented harnesses
- composition studies
- latency / wall-clock benchmarking
- sensitivity / tiering analysis
- research-side implementations and adapters

Do not put raw experiment results here.
Durable experiment outputs must land in `knlp-key-results` first.
