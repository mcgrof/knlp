========================================================
PHASE 1 — Accumulation Model (math + measurable quantities) (2–4h)
========================================================
1.1 Start from residual recursion:
x_{ℓ+1} = x_{ℓ} + f_{ℓ}(x_{ℓ})
Quantization induces:
x̃_{ℓ+1} = x̃_{ℓ} + f_{ℓ}(x̃_{ℓ}) + δ_{ℓ}(x̃_{ℓ})
Define e_{ℓ} = x̃_{ℓ} - x_{ℓ}.

1.2 Linearize:
e_{ℓ+1} ≈ (I + J_{ℓ}) e_{ℓ} + δ_{ℓ}
where J_{ℓ} = ∂f_{ℓ}/∂x.

1.3 Unroll:
e_{D} ≈ Σ_{ℓ=0}^{D-1} A_{ℓ→D} δ_{ℓ}
A_{ℓ→D} = Π_{j=ℓ+1}^{D-1} (I + J_{j})

1.4 Convert to norm bound:
||e_D|| ≤ Σ w_ℓ ||δ_ℓ||
where w_ℓ approximates amplification (product of operator norms).

1.5 Link representation error to loss:
Use a Lipschitz-style bound or empirical calibration:
ΔNLL ≈ c * ||e_D||  (fit c empirically)
or (better):
ΔNLL ≈ Σ α_ℓ ||δ_ℓ|| with α_ℓ = c*w_ℓ  (fit α_ℓ directly)

========================================================
PHASE 2 — Derive k(D, ε) Lower Bound (2–4h)
========================================================
Assume two noise levels:
- protected layers (INT8 or “high precision”): δ ~ σ8
- unprotected layers (INT4): δ ~ σ4   with σ4 >> σ8
Let D = #layers, k = #protected, and define effective weighted sum:
S(k) = Σ_{ℓ in unprotected} α_ℓ σ4 + Σ_{ℓ in protected} α_ℓ σ8

Requirement for PASS at tolerance ε:
S(k) ≤ B(ε)
where B(ε) is an error budget tied to ε (fit from data).

Worst-case bound (conservative):
Let α_max be max α_ℓ among unprotected layers.
Then:
(D-k) α_max σ4 + k α_min σ8 ≤ B(ε)
Solve for k:
k ≥ (D α_max σ4 - B(ε)) / (α_max σ4 - α_min σ8)

More realistic bound:
Sort layers by α_ℓ descending (importance weights).
Protect top-k weights.
Then:
Σ_{i=k+1..D} α_(i) σ4 + Σ_{i=1..k} α_(i) σ8 ≤ B(ε)
This yields an implicit lower bound for k (compute smallest k satisfying inequality).

Key theoretical “O(1)” condition:
If the tail sum of α_(i) is bounded as D grows (e.g. α decays fast, or only O(1) “sink” layers dominate),
then k* can remain O(1) for fixed ε.

========================================================
PHASE 3 — Estimate σℓ and αℓ Empirically (W7900) (4–10h)
========================================================
We need measurable proxies for:
- per-layer injected noise magnitude σℓ (from quant)
- per-layer amplification/importance αℓ

Use TWO complementary estimation routes:

Route A (direct, uses oracle ablations):
A1) For each layer ℓ:
- quantize KV at layer ℓ only (others high precision)
- measure ΔNLL (or ΔPPL) at L=8192 (and optionally 16384)
Define oracle_sensitivity sℓ = ΔNLL_ℓ
This sℓ is effectively αℓ σ4 (up to constant).

Route B (amplification traces via small noise injection):
B1) Inject a tiny Gaussian/uniform noise to KV at layer ℓ (one layer at a time),
small enough to remain in linear regime.
B2) Measure layerwise propagation:
- compute ||e_{j}|| at each subsequent layer j for a small batch (store traces)
Fit an average amplification factor w_ℓ (slope of log ||e_j|| vs j).
This gives w_ℓ, a proxy for A_{ℓ→D} magnitude.

Then define:
α_ℓ ≈ c * w_ℓ  (fit c by matching to oracle sensitivities)

Parameter fitting:
- Fit B(ε) by finding threshold on Σ α_ℓ σ that matches observed PASS/fail boundary from k-sweeps.
- Fit σ4/σ8 ratio from quantization step sizes or from comparing INT4 vs INT8 single-layer deltas.

Do this for BOTH models:
- 0.5B (D_small)
- 1.5B (D_med)
(Optional 3B if feasible)
