## 1. Core Objective (v7)

BPA v7 must answer **one question**:

> Does BPA + RA provide *stable, information-aligned KV scaling* as context length grows, or is the observed β≈0.25 an artifact of budget heuristics, layer choice, or evaluation range?

Everything below exists to pressure-test that.

---

## 2. Stress-Test the Weakest Assumption

**Weakest assumption in the current stack:**
That *RA_value scores correlate with future usefulness*, not just recency-biased salience.

### Required tests:

1. **Adversarial RA ablation**

   * Freeze RA_value scores from early tokens.
   * Continue generation with distribution shift (topic change).
   * Measure:

     * PPL degradation vs normal RA
     * β drift vs recency

2. **Temporal decorrelation test**

   * Shuffle far-chunk RA scores within fixed windows.
   * Preserve marginal distribution, break alignment.
   * If performance collapses → RA signal is semantic.
   * If not → RA is acting as a proxy for something simpler.

3. **Cross-layer RA corruption**

   * Inject noise into RA scores at *non-surgical* layers.
   * Confirm that only FIM-guided layers matter.

---

## 3. Dynamic Budget Experiment (Most Important v7 Addition)

Implement **query-conditional far budget**:

```
b(x_t) ∈ [0, B_max]
```

### Variants:

* Constant (v6 baseline)
* Linear in entropy
* Linear in attention concentration
* Learned scalar (1-layer MLP, gated behind flag)

### Metrics:

* β_eff(L) for L ∈ {512, 1024, 2048, 4096 if possible}
* Mean / p95 / p99 KV_kept
* PPL at fixed compute

**Hard rule:**
If learned budget does not reduce β or PPL simultaneously, drop it.

---

## 4. Collapse RA into Gate Features (No New Attention)

RA must *not* exist as a parallel attention path.

Implement these **cheap RA-derived features only**:

* Cached inbound RA mass per chunk (EMA)
* Rank-only RA indicators (top-k flags)
* Layer-aggregated RA scores (no per-head ops)

Compare against:

* Recency
* Random
* Full RA_value (v6)

Measure:

* Gate overhead
* PPL delta
* β delta

- L={512, 1024, 2048}

Stress test results at L=2048:

  stress_control    PPL=279.9+/-12.2 [267.7, 292.1]  (normal RA)
  stress_corrupt    PPL=280.0+/-11.5 [268.5, 291.5]  (noise injection)

Finding: Frozen RA (delta=+0.3) and shuffled RA (delta=+1.3) show

### v7 Verdict

What survived: RA_value beats recency by 11.5 PPL at L=2048
and this advantage is robust to RA manipulation (frozen/shuffled/
corrupt). Cheap RA features (EMA, rank) are sufficient.

What broke: RA signal is not semantically aligned — shuffling
RA scores causes only +1.3 PPL degradation. RA is a refined
popularity measure, not content-addressable retrieval. Dynamic
budget provides zero benefit and is dropped.

What remains uncertain: Whether beta=0.25 is a genuine BPA
property or an artifact of the local_window floor. L=4096
testing blocked by position embedding limits.
